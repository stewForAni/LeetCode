# 只出现一次的数字
给定一个非空整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。
说明：你的算法应该具有线性时间复杂度。 你可以不使用额外空间来实现吗？

### 集合
### HashMap
### 空间复杂度都是O(n)，需要额外的空间

---

### 位运算（异或：同为0，异为1）
```java
class Solution {
    public int singleNumber(int[] nums) {
        int t = 0;
        for (int i : nums) {
            t = t ^ i;
        }
        return t;
    }
}
```
### 时间复杂度都是O(n)
### 空间复杂度都是O(1)
