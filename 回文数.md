# 回文数

给你一个整数 x ，如果 x 是一个回文整数，返回 true ；否则，返回 false 。
回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。
例如，121 是回文，而 123 不是。

### 注意（0-9是回文数）

```java
class Solution {
    public boolean isPalindrome(int x) {
         if (x < 0 || x % 10 == 0 && x != 0) {
            return false;
        }

        char[] s = String.valueOf(x).toCharArray();
        for (int i = 0; i < s.length / 2; i++) {
            if (s[i] != s[s.length - 1 - i]) {
                return false;
            }
        }
        return true;
    }
}
```
